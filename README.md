# Fancy Interactive Resume

A lightweight animated **single-page resume** with GSAP-powered entrance effects, a theme switcher, and a shimmering header animation.

**[▶ Live Demo](https://woodyhoko.github.io/cv_minimum)**

---

## Features

- ✨ **GSAP animations** — sections slide and fade in on load
- 🌗 **Theme switcher** — toggle between light and dark modes
- 💫 **Shimmer header** — CSS keyframe animation on the header band
- 🗂️ **Structured sections** — Skills, Experience, Education, Projects
- 📦 **Zero dependencies at runtime** — GSAP loaded from CDN, single HTML file

---

## Stack

| Library | Purpose |
|---|---|
| [GSAP 3.11](https://gsap.com/) | Animation timeline |
| Vanilla CSS | Theme variables + layout |

---

open index.html
